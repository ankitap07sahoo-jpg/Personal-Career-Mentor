# AI Career Mentor Landing Page

A responsive single-page landing site for an AI-powered career mentoring product. Built with vanilla HTML, CSS, and JavaScript.

## Features
- Hero section with product messaging and stats
- Feature grid, career paths, and dashboard preview
- Skill gap and resume analyzer sections
- Scroll reveal animations

## Project Structure
- index.html
- styles.css
- script.js

## How to Run
Open index.html in your browser.

## Customization
Edit colors and styling in styles.css.
